# 📊 Software Engineering

Neste repositório está sendo desenvolvido projetos para aperfeiçoamento deste portifólio

---

## 🟢 Nível Iniciante — Fundação do Portfólio

### 1. Analisador de CSV com Pandas
Script que carrega datasets públicos, realiza limpeza, análise exploratória e gera visualizações.

### 2. Web Scraper de Notícias
Coleta automatizada de notícias (título, link, data) usando BeautifulSoup.

### 3. Dashboard Interativo com Streamlit
Aplicação web que transforma dados em dashboards interativos.

---

## 🟡 Nível Intermediário — Destaque Técnico

### 4. API REST de Previsão
Modelo de machine learning exposto via API (FastAPI/Flask).

### 5. Sistema de Recomendação
Recomendação baseada em similaridade entre usuários ou itens.

### 6. Análise de Sentimentos (NLP)
Classificação de textos como positivo/negativo.

### 7. Pipeline de Dados Automatizado
Pipeline que coleta dados de APIs, processa e salva em banco SQLite.

---

## 🔴 Nível Avançado — Diferencial

### 8. Chatbot com LLM
Assistente baseado em modelos de linguagem.

### 9. Detector de Anomalias Financeiras
Identificação de padrões incomuns em dados financeiros.

### 10. Projeto Integrador — ML End-to-End
Pipeline completo com ingestão, modelo, API e dashboard.
